# Coffee Shop Website

A modern, responsive coffee shop website built with React, Vite, and Tailwind CSS. Featuring a sleek hero section with custom fonts and a focus on craft coffee experiences.

## About

"Every Cup Begins a Story." Since 2025, born from a passion for craft and simplicity, we design coffee moments that connect people and inspire calm in the everyday rush.

### Key Features

- **Hero Section**: Stunning full-screen hero with background image and overlay
- **Brand Story**: Highlighting our commitment to quality coffee and community
- **Statistics**: Showcasing 100+ cups brewed, 50+ local farmers, and 3K+ customers served
- **Responsive Design**: Optimized for all devices using Tailwind CSS
- **Custom Fonts**: Novar and Ranade fonts for premium typography
- **Interactive Buttons**: Smooth hover effects and transitions

## Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite (with Rolldown)
- **Styling**: Tailwind CSS
- **Fonts**: Custom web fonts (Novar, Ranade)
- **Linting**: ESLint with React hooks and refresh plugins

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or bun package manager

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd coffee-shop
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   bun install
   ```

3. Start the development server:

   ```bash
   npm run dev
   # or
   bun run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
# or
bun run build
```

### Preview Production Build

```bash
npm run preview
# or
bun run preview
```
